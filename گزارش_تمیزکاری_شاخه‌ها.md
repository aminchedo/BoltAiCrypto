## 🔧 دستورات نگهداری

برای تمیزکاری آینده از این دستورات استفاده کنید:

```bash
# لیست شاخه‌های ادغام شده
git branch -r --merged main | grep -v "HEAD" | grep -v "main$"

# حذف شاخه ریموت ادغام شده
git push origin --delete branch-name

# پاکسازی مراجع محلی
git fetch --prune

# لیست همه شاخه‌های ریموت
git branch -r
```
